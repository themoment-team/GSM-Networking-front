# GSM Networking

광주소프트웨어마이스터고등학교 네트워킹 서비스입니다.

## Commands

```bash
$ git clone https://github.com/themoment-team/GSM-Networking-front.git
$ cd GSM-Networking-front
$ pnpm install

# run dev
$ pnpm dev

# run build
$ pnpm build

# run storybook
$ pnpm storybook
```
